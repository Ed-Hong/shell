## obsidian
`f2` rename file
`ctrl s` excalidraw save
`cmd m` toggle kanban/markdown mode
`f12` open sync log
`f10` open file history
`cmd shift p` open command palette _(same as vscode)_
### ui
`cmd h` hide all
`cmd j` hide left sidebar
`cmd l` hide right sidebar
`cmd k` hide tab bar
### editor
`cmd e` toggle preview/src mode
`cmd r` toggle reading/edit mode
`opt p` pin tab
`cmd \` split right
`cmd shift \` split down
`cmd shift [` fold more *(aka `cmd opt [`)*
`cmd shift ]` fold less *(aka `cmd opt ]`)*
### markdown
`opt [` insert markdown link [...](...)
`opt x` toggle checkbox [x]
`cmd shift .` toggle ~~strikethrough~~
`cmd shift x` toggle ~~strikethrough checkbox [x]~~
`opt h` toggle ==highlight==
### navigation
`cmd [` focus tab group left
`cmd ]` focus tab group right
`opt t` navigate back (aka mouse button 4)
`opt b` navigate forward (aka mouse button 5)
### workflow
`cmd d` open today's daily note
`cmd y` open previous daily note
`cmd m` open next daily note ( #todo make this be `create daily note for tmrw`)
